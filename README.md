# Netflix Clone

A Netflix-inspired web application developed using Flask, MySQL, and AWS integration. This project replicates the core UI and backend functionalities of Netflix, including user authentication, video content management, and responsive design.

---

## Features

- User authentication system
- Netflix-style responsive UI
- Flask backend integration
- MySQL database connectivity
- AWS S3 support for media storage
- Dynamic content rendering

---

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- Flask

### Database
- MySQL

### Cloud Services
- AWS S3
